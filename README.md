This project will be a website with a few links to recipes. It will also showcase our introductory HTML skills. Later we'll apply our CSS skills to beautify it.

Breakdown of the HTML skills I will demonstrate:
- HTML boilerplate setup
- Use of HTML elements like headers, lists, images, etc.
- Linking to other pages using relative links.

Other skills that will be demonstrated:
- Creating a repo on Github and cloning onto a local machine
- Committing often and providing useful messages.
- Pushing and pulling the repo to Github/local machine.
